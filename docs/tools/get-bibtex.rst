get_bibtex
==========

Get a BibTeX citation entry for a paper.

Searches Google Scholar for the paper by title and returns its BibTeX record
as a plain string ready to paste into a ``.bib`` file.

Parameters
----------

.. list-table::
   :header-rows: 1
   :widths: 22 12 12 54

   * - Parameter
     - Type
     - Default
     - Description
   * - ``title``
     - ``str``
     - *(required)*
     - Title of the paper to get BibTeX for.

Return Value
------------

Returns a raw BibTeX string (``str``). Example output:

.. code-block:: bibtex

   @article{vaswani2017attention,
     title={Attention is all you need},
     author={Vaswani, Ashish and Shazeer, Noam and Parmar, Niki and ...},
     journal={Advances in neural information processing systems},
     volume={30},
     year={2017}
   }

Examples
--------

**Classic paper:**

.. code-block:: json

   {
     "title": "Attention Is All You Need"
   }

**Recent work:**

.. code-block:: json

   {
     "title": "Language Models are Few-Shot Learners"
   }

.. note::

   BibTeX entries are generated by the scholarly library from Google Scholar data.
   Field completeness varies by paper. Always verify the entry against the
   publisher's official citation before submitting to a journal or conference.
